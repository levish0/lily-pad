`content/<로케일>/`에 마크다운을 쓰면 페이지가 됩니다. 이 파일은 `content/ko/index.md`이고 `/ko/docs`에서 서빙됩니다.

::: tip
`lily-pad.config.ts`에서 사이트 제목, 네비게이션, GitHub 링크를 설정하세요.
:::

## 다음은

- `.md` 파일을 추가하면 사이드바가 자동으로 갱신됩니다.
- 폴더로 페이지를 묶고, `section` frontmatter로 그룹 제목을 붙이세요.
- `content/en/`에 번역을 두거나 `project.inlang/settings.json`에서 로케일을 추가하세요.
