Write markdown in `content/<locale>/`, and it becomes a page. This file is `content/en/index.md` — served at `/docs`.

::: tip
Edit `lily-pad.config.ts` to set your site's title, navigation, and GitHub link.
:::

## Where to go next

- Add pages as `.md` files — the sidebar updates automatically.
- Group pages in folders; title groups with the `section` frontmatter field.
- Translate pages into `content/ko/` (or add your own locales in `project.inlang/settings.json`).
