콘텐츠를 `:::` 펜스로 감싸면 콜아웃이 렌더링됩니다. 다섯 타입이 있습니다: `tip`, `info`, `note`, `warning`, `danger`.

```md
::: tip
독자가 놓치기 쉬운 유용한 정보.
:::
```

::: tip
독자가 놓치기 쉬운 유용한 정보.
:::

::: info
중립적인 배경 정보.
:::

::: warning
조심하세요 — 물릴 수 있습니다.
:::

::: danger
이 동작은 파괴적입니다.
:::
